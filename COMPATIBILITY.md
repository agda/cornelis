# Known compability issues

Agda is known to change its API now and again. Supporting multiple versions of
Agda in the same release is difficult and maintenance-heavy. If you are using
one of these versions, you may need to downgrade.

## Agda versions

- [v2.7.0](https://github.com/agda/cornelis/tree/v2.7.0)
- [v2.7.1](https://github.com/agda/cornelis/tree/v2.7.1)
- [v2.8.0](https://github.com/agda/cornelis/tree/v2.8.0)

## Neovim versions

- [nvim v0.12.*](https://github.com/agda/cornelis/tree/nvim-0.12-compat)
